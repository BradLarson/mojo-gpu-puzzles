## Development

1. Make sure to have `magic` installed

    ```bash
    curl -ssL https://magic.modular.com/ | bash
    ```

    or is updated

    ```bash
    magic self-update

    ```

2. Build the book and open [localhost:3000](http://localhost:3000)

    ```bash
    magic run book
    ```

3. Test solutions on a compatible GPU

    ```bash
    magic run tests
    ```

4. Format

    ```bash
    magic run format
    ```
